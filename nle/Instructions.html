Calcul du gain

Pour chacun des {{ C.NB_TARGETS }} exercices, le programme informatique va mesurer la distance, en valeur absolue, entre la cible et la valeur sur laquelle vous aurez positionné le curseur.
Par exemple si la cible est 12.2 et que vous positionnez le curseur sur 11.5 la distance est de 0.7.

Le programme informatique va ensuite calculer la distance moyenne sur les {{ C.NB_TARGETS }} exercices.
Votre gain pour ce jeu sera alors égal à {{ C.CONSTANTE }} € - {{ C.FACTEUR_DISTANCE }} € x distance moyenne.
