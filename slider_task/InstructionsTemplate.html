Dans cette tâche, vous allez devoir placer un curseur sur une valeur cible. La valeur cible est un nombre entier compris entre 0 et 100. Le curseur est correctement positionné lorsque la valeur du curseur est égale à la valeur cible.

Vous disposez de {{ time_limit }} pour valider au moins {{ C.REQUIRED_SUCCESS }} curseurs.
Si vous validez au moins ce nombre de curseurs, vous gagnez {{ C.PAYOFF_SUCCESS }} ECU et pouvez continuer l'expérience.
Si vous ne validez pas {{ C.REQUIRED_SUCCESS }} curseurs, l'expérience s'arrête pour vous, vous serez dédommagé(e) du forfait de déplacement.

Pour placer les curseurs vous pouvez utiliser la souris et les flèches du clavier.
