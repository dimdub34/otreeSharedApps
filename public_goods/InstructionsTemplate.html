Cadre général

Au début de l'expérience, le programme informatique va former aléatoirement des groupes de {{ subsession.players_per_group }} personnes. La composition de ces groupes restera identique pour toute l'expérience. Vous ne pouvez pas identifier les autres membres de votre groupe et ils ne peuvent pas vous identifier.

L'expérience comporte {{ C.NUM_ROUNDS }} périodes identiques. Au début de chaque période, vous disposez de {{ C.ENDOWMENT }} jetons que vous devez entièrement répartir entre deux comptes : votre compte individuel et un compte collectif commun à votre groupe.

Fonctionnement des comptes

Le compte Individuel

Chaque jeton que vous placez dans votre compte individuel vous rapporte 1 ECU (à vous seul). De même, si un autre membre place un jeton dans son compte individuel, cela lui rapporte 1 ECU (à lui seul).

Exemple : Si vous placez 5 jetons dans votre compte individuel, votre gain issu de ce compte sera de 5 ECU.
Le compte Collectif

Le compte collectif est commun aux {{ subsession.players_per_group }} membres du groupe. Chaque jeton placé dans le compte collectif rapporte {{ subsession.mpcr }} ECU à chaque membre du groupe.

Exemple : {{ instructions_exemple_compte_collectif }}

Calcul du gain

Votre gain pour la période est égal à la somme de votre gain issu du compte individuel et de votre gain issu du compte collectif.

Exemple : {{ instructions_exemple_gain }}
{{ if subsession.seuil > 0 }}
⚠️ Attention au seuil minimal

Si le total de jetons placés par votre groupe dans le compte collectif n'atteint pas le seuil minimal de {{ subsession.seuil }} jetons, alors le gain issu du compte collectif pour chaque membre du groupe est de 0 ECU. Dans ce cas, le gain de la période pour chacun se limitera au nombre de jetons placés sur son propre compte individuel.

{{ endif }}

Déroulement d'une période

À l'écran, vous saisissez uniquement le nombre de jetons placés dans le compte collectif. Les jetons restants sont automatiquement affectés à votre compte individuel.

Les décisions sont simultanées. Lorsque tout le groupe a validé, un écran récapitulatif s'affiche avec : vos placements, le total placé par le groupe dans le compte collectif, et votre gain de la période.

Vous aurez accès en permanence à un historique sous forme de tableau, reprenant les résultats des périodes passées.
