Dans ce jeu, vous devez décider du nombre d'injections d'air que vous envoyez dans un ballon. Initialement le ballon est dégonflé. Chaque injection d'air vous fait gagner {{ C.PAYOFF_PER_PUMP|cu }}, donc plus le ballon est gonflé plus vous gagnez d'argent, sauf si le ballon explose, dans ce cas vous perdez tout.

Le ballon supporte au maximum {{ C.MAX_PUMPS }} injections, mais il peut exploser à n'importe quelle injection : 1ère, 2ème, ..., {{ C.MAX_PUMPS }}ème.
Concrètement, le programme informatique choisit au hasard un nombre compris entre 1 et {{ C.MAX_PUMPS }}, ce nombre détermine l'injection à laquelle le ballon explose. Chaque nombre a la même chance d'être tiré au sort.

Pour fixer le nombre d'injections d'air que vous souhaitez envoyer dans le ballon, vous devez utiliser les boutons et . Vous pouvez cliquer sur les boutons ou les maintenir appuyer.

Une fois votre décision prise, cliquez sur le bouton Valider. Vous verrez alors si le ballon explose ou non. Si le ballon explose votre gain est de {{ 0|cu }}, sinon vous gagnez {{ C.PAYOFF_PER_PUMP |cu }} par injection.
